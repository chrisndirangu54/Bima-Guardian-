# Final Filled Quote Output (Example)

This is what the generated quote PDF contains after admin uploads a company quote template and a company rate card.

## Header block (always included)
- Quote title (from `quote_template.title` if available, else `Insurance Quote`)
- Quote ID
- Policy Type
- Policy Subtype
- Company
- Premium (KES)
- Generated timestamp

## Company template section (optional, from uploaded quote template)
For each section item:
- `label`
- resolved value from `fieldKey`
- optional `prefix` + `suffix`

Example uploaded quote template section:
```json
{
  "label": "Premium",
  "fieldKey": "premium",
  "prefix": "KES ",
  "suffix": ""
}
```

Example rendered line in the PDF:
- `Premium: KES 12500.00`

## Details block (always included)
All captured form fields are printed as key/value lines.

## Footer (optional)
If `quote_template.footer` is provided, it is appended at the bottom.

## Exact same visual mode
If `useOriginalLayout=true` and `pdfTemplateKey` is provided in the company quote template,
the generated quote preserves the uploaded document's look/structure/format and only overlays
calculated values at configured coordinates.
